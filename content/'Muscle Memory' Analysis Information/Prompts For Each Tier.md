##### 1. Tier 1 Test - 'Blind' Data:
- You are an expert data analyst. Each row of the dataset below is one subject
  recorded over one period; the remaining columns are numerical measurements.
  1. Study the data and report your conclusions.
  2. Propose 2–3 hypotheses about what could account for what you find.
  3. What additional data would help you test those hypotheses?
- The guise of the 'expert data analyst' is set up to guide the response without introducing the bias of, for example, saying 'expert movement ecologist'. The set up in the next two tiers answers this without bias in this first instance. The following instructions are then established to ensure the result is given in the right structure without influencing the result itself.
##### 2. Tier 2 Test - Visible Data + True Animal:
- Exact same initial prompt with the addition of a follow up: The subjects are individual GPS-collared female elk (Cervus canadensis), tracked in and around Banff National Park in the Canadian Rocky Mountains. Interpret the data as a movement ecologist would.
- The idea of adding this contribution is to compare whether each LLM were using the data in front of them, or their own muscle memory.
##### 3. Tier 3 Test - Visible Data + False Animal:
- Same test as tier 2 but this time the LLMs are told the subjects are roe deer in the Britain: The subjects are individual GPS-collared female roe deer (Capreolus capreolus), tracked across lowland Britain. Interpret the data as a movement ecologist would.
- The idea behind this is the same as tier 2. It just helps seperate a real analysis from one that is made up based on the AI's training.
- The choice of roe deer was quite an easy one as they are a species that are highly resident and share a different genus to the Canadian Elk in the study. The LLMs should pick up on the large distances traveled being odd for this species.
